# FAQ

#### Does my favorite VCV Rack module run on the MetaModule?

Currently over 800 modules are available, and the list is growing! See [Plugins](https://metamodule.4ms.info/plugins) if what you're looking for is already ported to the MetaModule platform. If not, you can request the developer or maintainer port it, or make a request on our [forum](https://forum.4ms.info).
